Sobre mim

Tenho 28 anos, nasci e moro em São Paulo- SP, sempre fui curioso e gostava de criar coisas novas, isso me levou a engenharia e posteriormente a programação onde me encontrei de vez, criei este site com o intuito de falar um pouquinho sobre temas e assuntos que gosto, além de melhorar minhas habilidades como desenvolvedor.

Como funciona a plataforma?

Para postar algo, basta se cadastrar e em sequencia logar, assim você terá acesso tanto a escrever novos posts e novos temas, quanto pesquisar por assuntos já escritos, através da barra de pesquisa.

Como foi criada a plataforma?

Foi criada como uma atividade de desenvolvimento web full stack da ong (Generation Brasil) de formação de novos desenvolvedores júnior para o mercado de trabalho, esta plataforma utiliza ferramentas como Angular e Bootstrap para o desenvolvimento do front end, e Java, Spring boot e MySql para o back end e o banco de dados, vale ressaltar que essa plataforma possui uma camada de segurança onde para acessar o feed é necessario estar logado, e todas as senhas cadastradas são criptografadas.
